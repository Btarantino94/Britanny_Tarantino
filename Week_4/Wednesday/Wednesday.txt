Git daily challenges

1. Write the command that tells you whether there have been any changes done to your repository that you have not yet pulled.

git status - Checks the status of repo

2. Suppose you are currently in the master branch of your repository. Create a new branch called calculator and switch into it.

git checkout -b calculator


3. Write the command needed to list all the branches in your current branch.

git branch to list the branches.



4. Suppose there is a branch named clownNose in your current branch. Write the command necessary to merge clownNose with your current branch.

git merge clownNose


5. Write the command to create a new repository named myRepo

git init myRepo 
  git init is the command to initialize a git myRepo.
